## Проекты

### 1. ИИ-агент для анализа медицинской статистики Санкт-Петербурга через Text-to-SQL (Хакатон)

Веб-приложение, которое позволяет получать аналитику по медицинским данным Санкт-Петербурга через текстовые запросы на естественном языке. Пользователь задаёт вопрос — система преобразует его в SQL-запрос, выполняет его в DuckDB и возвращает ответ в виде текстового отчёта с интерактивными графиками. Встроенный механизм самокоррекции перехватывает и исправляет ошибочные запросы для повышения надёжности.

**Стек:** Python, LangChain, OpenRouter API, DuckDB, Streamlit, Plotly

**Репозиторий:** [https://github.com/hallienpng/AI-for-Finance](https://github.com/hallienpng/AI-for-Finance)

---

### 2. RAG Financial Assistant (Хакатон)

Система для ответов на финансовые вопросы, построенная по RAG-архитектуре. Документы индексируются с помощью FAISS, поиск дополняется модулем ранжирования (rerank), а финальный ответ генерируется LLM. Проект включает полный пайплайн от обработки базы знаний до генерации ответов в формате, готовом для хакатонных сабмитов.

**Стек:** LangChain, FAISS, OpenAI API

**Репозиторий:** [https://github.com/ViacheTix/medhakathon](https://github.com/ViacheTix/medhakathon)
